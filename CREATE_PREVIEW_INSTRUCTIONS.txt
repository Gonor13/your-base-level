# Для создания preview.png:

1. Откройте https://www.canva.com/
2. Создайте дизайн 1200x630px
3. Текст: "Your Base Level"
4. Подтекст: "Discover your Base network status"
5. Сохраните как preview.png
6. Загрузите в папку public/

ИЛИ используйте онлайн генератор:
https://ogimage.dev/

ИЛИ создайте через Paint:
- Откройте Paint
- Файл → Свойства
- Ширина: 1200, Высота: 630
- Залить синим цветом (#0052FF)
- Добавить текст белым цветом
- Сохранить как preview.png
